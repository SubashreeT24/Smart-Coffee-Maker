## 📖 1. Overview

**Smart Coffee Maker** is a fully embedded, automated coffee brewing system built around the **8052 microcontroller**. It detects a cup automatically using an IR sensor, lets the user choose a brewing mode via push buttons, and dispenses coffee powder and sugar using motor-driven mechanisms — guided throughout by an LCD display for user interaction.

---

## ✨ 2. Features

- 👀 Automatic cup detection using IR sensor
- 🔄 Ingredient dispensing (coffee powder & sugar) using dual DC motor control
- 🔘 Push button controls for mode selection
- 🖥️ LCD display for guided, user-friendly operation
- ⚙️ Embedded C based microcontroller programming
- 🔧 Complete hardware integration and testing across sensors, motors, and display

---

## ☕ 3. Modes Available

| Mode | What it does |
|---|---|
| **Coffee Strength Selection** | Lets the user choose the quantity of coffee powder dispensed |
| **Sweetness Level Selection** | Lets the user choose the quantity of sugar dispensed |
| **Automatic Mode** | Dispenses a predefined, fixed quantity of coffee powder and sugar with no manual input |

---

## 🛠️ 4. Components Used

| Component | Purpose |
|---|---|
| **8052 Microcontroller** | Core controller — handles sensor input, motor control, and LCD output |
| **IR Sensor** | Detects presence of a cup before brewing starts |
| **DC Motors (x2)** | Controls the flap mechanism and ingredient (powder/sugar) dispensing |
| **LCD Display** | Guides the user through mode selection and brewing status |
| **Push Buttons** | Used for selecting brewing mode and adjusting quantities |
| **Motor Driver Circuit** | Drives the DC motors from microcontroller signals |
| **Power Supply Unit** | Powers the microcontroller and connected hardware |

---

## ⚙️ 5. How It Works

1. **Cup Detection** — The IR sensor detects when a cup is placed in position
2. **Mode Selection** — The user selects a mode (Strength / Sweetness / Automatic) via push buttons, guided by LCD prompts
3. **Ingredient Dispensing** — Based on the selected mode, the DC motors control the flap and dispense the appropriate quantity of coffee powder and/or sugar
4. **Status Display** — The LCD shows real-time status throughout the brewing process
5. **Completion** — Once dispensing is complete, the system resets and waits for the next cup
