# Smart Coffee Maker using 8052 ☕

A smart automated coffee maker built using an 8052 microcontroller with sensor-based control, multiple brewing modes, and motor-driven ingredient dispensing.

## 🔹 Highlights

- Built using an 8052 microcontroller ⚙️
- Implemented IR sensor-based cup detection 👀
- Added push button controls for mode selection 🔘
- Used dual DC motors for flap and ingredient control 🔄
- Integrated LCD display for user interaction 🖥️
- Designed and tested multiple coffee preparation modes ☕

## ☕ Modes Available

- Strong Coffee
- Sugar Level Selection
- Automatic Mode

## 🔗 Features

- Automatic cup detection using IR sensor
- Ingredient dispensing using motor control
- User-guided operation through LCD display
- Embedded C based microcontroller programming
- Complete hardware integration and testing

## 🛠️ Components Used

- 8052 Microcontroller
- IR Sensor
- DC Motors
- LCD Display
- Push Buttons
- Motor Driver Circuit
- Power Supply Unit
